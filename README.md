# ML_tutorial

딥러닝(Deep Learning) 위주 실습 및 학습용 저장소입니다.

- 주요 내용: 회귀, 분류, 신경망, 실습 노트북
- 예제: MNIST, 회귀 분석, 3D 시각화 등

## 폴더 구조
- `src/` : 실습 코드 및 노트북
- `data/` : 데이터셋

## 사용법
1. Python 가상환경 생성 및 활성화
2. requirements.txt 또는 노트북 내 패키지 설치
3. Jupyter Notebook 또는 VS Code에서 실습
